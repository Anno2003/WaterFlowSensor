# Firmware Water Flow Sensor Sederhana

## Penggunaan
1. clone firmware ini dari repositori
`git clone git@github.com:Anno2003/WaterFlowSensor.git waterflow_firmware`
2. buka menggunakan arduino ide lalu flash ke perangkat

> ![NOTE]
> firmware ini ditujukan untuk esp32 s3
> dicompile menggunakan arduino IDE 2.3.6 dengan esp32 core versi 3.3.3 

## Dependency
- [WifiManager 2.0.17](https://github.com/tzapu/WiFiManager.git)
- [AsyncMQTT_Generic 1.8.1](https://github.com/khoih-prog/AsyncMQTT_Generic)
- [ESP Async WebServer 3.8.1](https://github.com/ESP32Async/ESPAsyncWebServer)
- [RemoteDebug2 4.0.1](https://github.com/karol-brejna-i/RemoteDebug.git)
- [Adafruit SSD1306 2.5.15](https://github.com/adafruit/Adafruit_SSD1306)
